# 의뢰인 미팅 준비 초간단 체크리스트

- 작성일: 2026-07-13
- 목적: 짧게 보고 바로 실행하는 미팅 준비 문서

---

## 1. 준비물 (이것만 챙기면 됨)

1. 노트북 + 충전기
2. 현재 시안 링크 또는 로컬 실행 화면
3. 메모 도구 (노션/종이)
4. 일정표 (다음 미팅 날짜 잡기용)

선택 준비물
- 웹사이트 인쇄본 1~2장: 필요하면 챙김
- 인쇄가 꼭 필요하지는 않음. 화면으로 보여줘도 충분함.

---

## 2. 문서 준비 (이번 미팅 기준)

필수
1. 미팅 메모 템플릿 (질문/답변 적는 용도)
2. 계약/지급 주체 확인 메모 칸
- 계약자: 최해진 교수님(개인)인지, 중앙대학교(기관)인지
- 비용 지급자: 누구 계정/예산에서 지급되는지
- 세금계산서(또는 영수증) 발행 대상: 개인/기관 중 누구인지

있으면 좋은 것
1. 간단 견적 범위표 1장
- 금액 확정이 아니어도 됨
- "어디까지 하면 비용이 얼마나 달라지는지"만 보여주면 충분

2. 계약서 초안
- 이번이 첫 방향 미팅이면 없어도 됨
- "다음 미팅에서 계약서 검토"로 잡아도 괜찮음

정리
- 견적서/계약서는 이번 미팅에서 반드시 필요한 문서는 아님
- 대신 "범위와 운영 방식"을 먼저 확정하는 것이 더 중요

---

## 3. 꼭 물어볼 질문 (핵심만)

1. 지금 웹사이트는 누가 관리하고 있나요?
2. 지금 배포는 어떻게 하고 있나요?
3. 이번 프로젝트 계약자는 누구인가요?
- 최해진 교수님 개인 계약인지, 중앙대학교 기관 계약인지
4. 비용은 누구에게 청구하면 되나요?
- 실제 지급 주체(개인/학교/산학협력단) 확인
5. 세금계산서(또는 영수증)는 누구 명의로 발행해야 하나요?
6. 앞으로 운영은 누가 맡을 예정인가요?
7. 업데이트를 자주 하는 항목은 무엇인가요?
- 예: 뉴스, 논문, 멤버
8. 운영 방식은 어떤 쪽이 편한가요?
- 개발자 수정 방식
- 문서/시트 입력 방식

---

## 4. 이번 미팅에서 결정할 것

1. 리뉴얼 1차 범위 (어디까지 먼저 할지)
2. 계약 주체 (개인/기관)
3. 지급 주체 + 증빙 발행 대상 (세금계산서/영수증 명의)
4. 운영 담당자 (실제 수정할 사람)
5. 배포/관리 방식 (현재 유지 vs 변경)
6. 다음 단계 일정 (시안 확정일)

---

## 5. 미팅 끝나고 바로 할 일

1. 오늘 결정된 내용 5줄 요약 공유
2. 못 정한 항목 따로 표시
3. 다음 미팅 날짜/목표 확정

---

## 한 줄 요약

이번 미팅의 목표는 "예쁜 화면 설명"보다, "누가 어떻게 운영할지"를 확정하는 것이다.
